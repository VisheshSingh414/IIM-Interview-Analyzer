## 🧠 Overview  
The **IIM Interview Analyzer** is a Generative AI-based web application that evaluates interview responses and provides intelligent feedback. Built with Streamlit, it offers an interactive interface for users to input responses and receive instant AI-driven insights.

---

## 💼 Business Problem  
Interview evaluation is often subjective and inconsistent. This project aims to:

- Standardize feedback using AI  
- Reduce human bias in interviews  
- Provide instant and structured evaluation  
- Help candidates improve performance  
- Enable scalable interview assessment systems  

---

## ⚙️ How It Works  

1. 🎤 User enters interview response in Streamlit app  
2. 🤖 LLM processes and analyzes the response  
3. 🧠 Evaluates:
   - Content quality  
   - Clarity  
   - Structure  
   - Relevance  
4. 📊 Generates feedback with improvement suggestions
